\section{数学公式}
以下是一些基本的数学公式：

\subsection{二次方程}
二次方程的一般形式是 \( ax^2 + bx + c = 0 \)，其解可以通过求根公式得到：
\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

\subsection{欧拉公式}
欧拉公式是一个在复分析领域的公式，它将三角函数与复数指数函数联系起来：
\[ e^{i\theta} = \cos(\theta) + i\sin(\theta) \]

\subsection{积分}
积分是微积分中的一个重要概念，一个常见的积分公式是：
\[ \int_{a}^{b} f(x) \, dx \]
